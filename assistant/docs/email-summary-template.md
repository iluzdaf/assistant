# Email Summary Template

- Include enough context that the email is understandable without reopening it.

```md
# Email summary YYYY-MM-DD

- Date: YYYY-MM-DD
- Scope: Emails triaged in this run that did not need review

## Summary

- Subject: Short subject
  - From: sender name
  - Date: YYYY-MM-DD
  - What it is: Invoice, update, receipt, newsletter, notification, or other clear category
  - Key details: The main facts, request, deadline, amount, confirmation number, or change

- Subject: Another subject
  - From: sender name
  - Date: YYYY-MM-DD
  - What it is: ...
  - Key details: ...

## Notes

- Anything worth remembering later, such as a follow-up date or a useful reference number.

#email #fyi
```
